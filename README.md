# GROKIPEDIA v2.5 — Freedom Vortex PWA

**Installable • Offline • Mobile-only • Bias meters • Live scrape**

> Add to home screen → real app. Works offline. Results cached forever.

## Install
- Android: Chrome → "Add to home screen"  
- iPhone: Safari → Share → "Add to Home Screen"

## Features
- Dual boxes (X #Grokipedia + web)  
- Bias meter on every result  
- RSS X scrape (never blocked)  
- Offline cache + last search  
- <60 KB total

Tweet `#Grokipedia anything` → appears for everyone.
