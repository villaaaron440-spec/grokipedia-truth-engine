# Grokipedia

**Real-Time Truth Engine** – A journalist-grade search tool that scans, analyzes, and links to primary sources with bias transparency.

Live at: [https://villaaaron440-spec.github.io/Grokipedia-/](https://villaaaron440-spec.github.io/Grokipedia-/)

---

## Features

- **Instant Search** across news, policy, and data
- **Bias Detection** (Left / Center / Right)
- **Direct Source Links** – no paywalls, no spin
- **Tech-Futuristic UI** – Orbitron + glowing effects
- **Mobile Ready**

---

## How to Use

1. Type your question
2. Press **Activate Truth Engine**
3. Get:
   - Source list
   - Bias breakdown
   - Fact check

---

## Tech

- HTML5 + CSS3 (Grid, Animations)
- Vanilla JS
- Google Fonts: Orbitron + Roboto Mono
- No frameworks. No bloat.
